Syluxent System Documentation Package

Contents:
- Syluxent_System_Architecture_Documentation.docx
- Syluxent_System_Features_Presentation.pptx

Purpose:
This folder is a movable capstone documentation package. The Word document explains
the system architecture from frontend to backend, including Flask, SQLAlchemy,
Supabase Postgres, Render deployment, GitHub source control, security, workflows,
and recommended future checks. The PowerPoint deck summarizes the system, its
features, architecture, deployment model, database model, and reliability roadmap.

Notes:
- No database password or secret value is included.
- The architecture reflects the current Supabase-only Postgres design.
- DOCX visual rendering could not be completed because LibreOffice/soffice was
  not available in this Windows environment; structural checks were completed.
- PPTX preview rendering and contact-sheet QA were completed.
